# Hell Yeah or No — Reading List

A distraction-free table of contents for [Hell Yeah or No](https://sive.rs/n) by Derek Sivers. No reviews, highlights, or purchase links — just the chapters so you can focus on reading.

**Live:** [tinhhuynh.github.io/hellyeahandno](https://tinhhuynh.github.io/hellyeahandno)

## Features

- All 67 chapters grouped by section (Opening, Updating Identity, Saying No, etc.)
- Each chapter links to the original essay on [sive.rs](https://sive.rs)
- Check off chapters as you read; progress is saved in your browser
- Progress bar and dark mode support

## Local

Open `index.html` in a browser. No build step.
